# Dino POS Landing Page — Redesign Plan

## ปัญหาที่จะแก้
1. เว็บดูเหมือน template — ทุก section คล้ายกัน ไม่มีจุด wow
2. ไม่มี Hero image/mascot เป็นพระเอก
3. สีเขียวไม่ตรงกับแบรนด์ม่วง
4. Product โผล่ช้าเกินไป
5. ไม่มี story/character

## โทนสีใหม่ (Purple Brand)
```
Primary:    #7C3AED  (violet-600)
Secondary:  #A855F7  (purple-500)
Background: #FAFAFA  (neutral-50)
Dark:       #1E1B4B  (indigo-950)
Accent:     #22C55E  (เขียว — ใช้นิดหน่อยตามจุดจำเป็น)
```

## Asset Strategy
- **Mascot:** ใช้ `<DinoCharacter>` component ที่รับ `variant` prop (`hero` | `pointing` | `carrying` | `giving` | `sad`) — ตอนนี้ render เป็น **div placeholder สีม่วง** (gradient + ตา + ปากเบื้องต้น) พร้อม structure ที่แทนที่ด้วย `<Image src="/dino-*.png">` ได้ง่ายในอนาคต
- **Icons:** ใช้ `lucide-react` (เดี๋ยวติดตั้งตอน implement)

## โครงสร้างใหม่ (ลำดับ storytelling)

```
1. Navbar          — sticky, glass effect, โลโก้ + menu + Download
2. Hero            — Dino mascot (placeholder) + headline + CTA (ม่วง)
3. ProductShowcase — โชว์ UI mockup ทันที (iPad/Desktop/Mobile frames)
4. StorySection    — scroll storytelling "ร้านค้าเล็ก ไม่ควรจ่าย 700/เดือน" → "จ่ายเฉพาะ Cloud" → "ทุกอย่างยังอยู่บนเครื่อง"
5. FeaturesSection — interactive grid พร้อม Dino ชี้
6. PricingSection  — FREE card (เด่น) + Add-ons card, Dino แจกของ
7. CTASection      — Download ใหญ่ + Dino ทักทาย
```

## ไฟล์ที่จะสร้าง/แก้ไข

### แก้ไข
- `app/globals.css` — เปลี่ยน color tokens เป็นม่วง, เพิ่ม animation ใหม่ (gradient shift, blob float, blink eyes)
- `app/layout.tsx` — ปรับ metadata, คง Thai font
- `app/page.tsx` — เปลี่ยนลำดับ section ตาม storytelling ใหม่

### สร้างใหม่ (แทนที่ของเดิมทั้งหมด)
- `app/components/DinoCharacter.tsx` — mascot placeholder รับ `variant` + `size` props
- `app/components/Navbar.tsx` — sticky navbar
- `app/components/Hero.tsx` — hero ใหม่ + mascot
- `app/components/ProductShowcase.tsx` — mockup UI หลาย device
- `app/components/StorySection.tsx` — scroll storytelling
- `app/components/FeaturesSection.tsx` — feature grid + Lucide icons
- `app/components/PricingSection.tsx` — FREE vs Add-ons
- `app/components/CTASection.tsx` — final CTA

### คงไว้
- `app/components/AnimateOnScroll.tsx` — reuse ระบบ animation เดิม (ทำงานได้ดีแล้ว)

### ลบ
- ไฟล์ section เดิมที่ไม่ใช้แล้ว: `DinoLogo.tsx`, `SectionWhy.tsx`, `SectionMission.tsx`, `SectionPhilosophy.tsx`, `SectionProduct.tsx`, `SectionFeatures.tsx`, `SectionPricing.tsx`, `SectionCTA.tsx`

## Animation & Polish
- ใช้ fade/scale/blur เดิมจาก AnimateOnScroll
- เพิ่ม: gradient background blob เคลื่อนไหว, mascot float + blink ตา, hover lift บน cards
- Scroll indicator ใน hero
- Sticky navbar ที่เปลี่ยนสีตอน scroll

## Acceptance criteria
- `bun run build` ผ่านไม่มี error
- โทนสีทั้งเว็บเป็นม่วง/ขาว/ดำ (ไม่มีสีเขียวเป็นหลัก)
- มี Dino mascot (placeholder) ปรากฏในหลาย section
- Product showcase อยู่ในลำดับต้น ๆ (หลัง hero)
- ใช้ Lucide icons แทน emoji
